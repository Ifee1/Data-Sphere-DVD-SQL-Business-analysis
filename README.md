# Data Sphere SQL Analysis

## Project Overview
This project analyzes the Maven Movies DVD rental database using SQL to explore store operations, inventory composition, customer behavior, and catalog coverage.
The goal of the analysis is to demonstrate how SQL can be used to extract meaningful business insights from relational databases.

## Dataset
The dataset represents the operations of a fictional DVD rental company and includes information on:

- Stores
- Inventory
- Films
- Customers
- Payments
- Actors
- Film categories
- Advisors and investors

## Key Analytical Questions
The analysis explores several operational and strategic questions including:

- Who manages each store and where are the stores located?
- What inventory is available across stores?
- How is inventory distributed across film ratings?
- What is the financial exposure of inventory replacement costs?
- Who are the most valuable customers?
- How well does the catalog cover award-winning actors?

Detailed questions can be found in:

**business_questions.md**

## SQL Techniques Used
This project demonstrates the use of several SQL techniques including:

- INNER JOIN
- LEFT JOIN
- Aggregations (SUM, COUNT, AVG)
- GROUP BY
- ORDER BY
- CASE statements
- Window functions (RANK, ROW_NUMBER)
- Bridge tables for many-to-many relationships
